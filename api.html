Api for Developers

En esta sección encontrarás los servicios Rest para registrar tu Robot en el Monitor

POST http://10.43.24.100:7001/RoboticRest/resources/robot/{idrobot}?action=init

Instrucción para iniciar el robot idrobot, retorna msg con el id que será usado durante el resto de la ejecución.


POST http://10.43.24.100:7001/RoboticRest/resources/robot/{idrobot}/{idrun}?step=enviar%20mail&value=abierto&result=ERROR

Instrucción para registrar una tarea realizada por el robot idrobot. Se registra step para la descripción de la tarea, value para los valores enviados y result para el resultado (OK o ERROR) de una ejcucion (idrun).


PUT http://10.43.24.100:7001/RoboticRest/resources/robot/{idrobot}/{idrun}?action=finish

Instrucción para finalizar la ejecución idrun del robotidrobot.
